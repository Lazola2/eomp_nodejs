Routes Description

Request Route Description
Clients
get / Route to this page (home page)
get /clients Requests data for all clients
get /client/:id Request data for a client with a matching id
post /login Logs the client
put /client/:id Updates a client record with a matching id
post /register Registers a new client
delete /client/:id Deletes a client record with a matching id
Vehicles
get /vehicles Requests data for all vehicles
get /vehicle/:id Registers data for a vehicle with a matching id
post /vehicle/:id Adds a new vehicle.
put /vehicle/:id Updates a record of a vehicle with a matching id.
delete /vehicle/:id Deletes a record of a vehicle with a matching id.
Purchases
get /purchases Requests data for all purchases
get /purchase/:id Requests data for a purchase with a clients's matching id
post /purchase/:id Creates a new purchase.
put /purchase/:id Updates a record of a purchase with a matching id.
delete /purchase/:id Deletes a record of a purchase with a matching id.
